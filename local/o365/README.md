# Microsoft 365 and Microsoft Entra ID Plugins for Moodle

## o365 Local Plugin

This plugin provides libraries and services and power other Microsoft 365 plugins. In addition to providing implementations of Microsoft 365 apis, this plugin handles a wide variety of Moodle events to fully integrate Microsoft 365 into a Moodle installation.

This is part of the suite of Microsoft 365 plugins for Moodle.

This repository is updated with stable releases. To follow active development, see: https://github.com/Microsoft/o365-moodle

## Installation

1. Unpack the plugin into /local/o365 within your Moodle install.
2. From the Moodle Administration block, expand Site Administration and click "Notifications".
3. Follow the on-screen instuctions to install the plugin.

For more documentation, visit https://docs.moodle.org/34/en/Office365

For more information including support and instructions on how to contribute, please see: https://github.com/Microsoft/o365-moodle/blob/master/README.md
